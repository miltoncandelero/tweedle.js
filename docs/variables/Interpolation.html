    • CatmullRom:function
      • CatmullRom(v: number[], k: number): number

      • A Catmullrom spline is a curve where the original set of points is also used as control points. Usually Catmullrom splines need two extra elements at the beginning and the end of the point set. This function contemplates that and doesn't need them.
